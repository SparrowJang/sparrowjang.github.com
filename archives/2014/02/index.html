The docker basic operation

What is the docker?

docker是一個能讓你的server中，擁有乾淨的環境，以及運行各種的linux平台，例如你開啟很多web service，可能有python、php、tomcat…，這時你可透過docker，讓這些web service，都擁有各自的環境，最後透過server的apache，proxy到各項服務，至於docker跟vm有什麼不同，可以參考這篇。
