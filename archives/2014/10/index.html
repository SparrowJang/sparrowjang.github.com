My github

angularjs

  • ngImageEditor
    載入一個圖片，讓user自己截圖。

  • ngFileReader
    用directive封裝的讀檔module，另外載入FileReader，也支援IE8+。(demo)

  • ngPreviewCover

  • ngHttpPool
    管控request在pool裡同時間的最大數量。

  • angularjs-placeholder
    讓不支援的html5的瀏覽器，也能使用placeholder。IE7+以上。

  • ngGoogleMap
    一個簡易版本的google map封裝。

  • ngPlaceSearchBox
    google map的place search box封裝，可以查詢需要的地址、地點。

javascript

  • spTimer
    一個倒數的計時器。

  • imageConverter
    依照size會自動擷取圖片中央的區塊。

  • mediaMonitor

  • frame-observer
    frame之間的溝通工具，擁有註冊event、method、promise功能。

  • string-form-utils
    全形半形之間互換。

jquery

  • jquery-uploader
    支援IE7做ajax upload使用，原理使用iframe。

polymer

  • spScrollView
    scroll裡面裝多個view，一次只顯示一個，可以互相切換，與android的scrollablevie功能相同。

python

  • image_converter
    與imageConverter相同，這個是後端使用。

docker

  • generator-ms-docker
    用來產生可以運行在docker的project。

  • docker hub
    封裝過後的docker images。
